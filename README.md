# Updated PDF Merger

A desktop Electron app for merging PDF files with reliable handling for very large inputs.

## Features

- Merge multiple PDF files into one output file.
- Drag-and-drop file picking and drag-to-reorder list control.
- Automatic safe-mode planning when command/input limits are high.
- Splits huge PDF files into smaller page segments so merges can push through very large sizes and path-length/argument limits.
- Progress updates during long merges.
- Includes bundled qpdf setup/usage for robust merge operations.
